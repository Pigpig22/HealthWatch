📱 ESP32 实时数据监控 App

一个基于 ESP32 + 移动端 App 的实时数据监测系统，通过蓝牙连接单片机设备，支持多通道传感器数据可视化、紧急呼叫功能，适用于健康监护、环境监测等场景。

主界面：数据列表 + 一键拨打预设紧急联系人
